Social Media Impact on Academic Performance

A comprehensive user study analyzing the correlation between short-form social media usage and academic performance among Emory University students. Designed and executed a survey methodology, collected data from 150+ students, and performed statistical analysis to identify patterns in media consumption and GPA outcomes.

User Research Data Analysis Survey Design Statistical Modeling
View Full User Study

Research Question

How does the frequency of social media use affect Emory students' academic performance?

Methodology

  • Surveyed 150+ Emory University students
  • Quantitative analysis of hours spent vs GPA
  • Qualitative assessment of self-perceived impact
  • Cohen's Kappa reliability measure (Îº = 0.72 - Satisfactory agreement)

Key Findings

  • Inverse correlation between social media hours and GPA (r = -0.42)
  • 63% of heavy users (4+ hrs/day) reported GPA below 3.4
  • 72% of respondents agreed they overuse social media
  • TikTok and Instagram accounted for 58% of total usage

Tools Used

Google Forms Python (Pandas, NumPy) Google Sheets Data Visualization

Our Design Process

We combine UX methodologies with technical implementation

1

Research

Conduct user interviews, competitive analysis, and technical feasibility studies to understand both user needs and implementation constraints.

2

Define

Synthesize findings into clear problem statements, user personas, and technical requirements documents.

3

Ideate

Generate solutions through design sprints, considering both user experience and technical architecture.

4

Prototype

Create interactive prototypes and technical proofs-of-concept to validate both design and implementation approaches.

5

Test

Conduct usability testing and technical performance evaluations to gather feedback for iteration.

6

Implement

Develop the final product with attention to both user experience details and code quality.
